Patron categories overview¶

Click here to see the “Patron categories” section of the Koha manual.

Patron categories define “types” of borrowers. They are one of the three elements that control circulation rules.

Only NEKLS staff can create or modify patron categories. Staff at member libraries that can see a summary of patron category data by running report 3517.
