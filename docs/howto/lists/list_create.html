How to create a list¶

Creating lists from the staff client¶

Creating an empty list¶

  1. From the home page, click on “Lists”

  2. Click on the button that says “+ New list”

  3. Give the list a name; set the sort order; mark the list as “Private” or “Public;” and set the “Allow changes …” permissions

  4. Then click on “Save”

  5. The new list should now appear in the list of lists in the Lists module

Create a list from search results¶

  1. Search the catalog (In this example I’m searching for the author “Ian Fleming”)

  2. Add checks to the checkboxes of the titles you want to add to the list you’re creating

  3. Click on “Add to list” from the toolbar and click on “New list” at the very bottom of the drop-down menu

  4. A pop-up window will appear – once you add a name and choose a category, click on “Save”

  5. The new list should now appear in the list of lists in the Lists module

Create a list from a title’s details page¶

  1. Search the catalog (In this example I’m searching for the author “Jane Austen,” the title “Sense and sensibility,” and the ISBN “9781676583387”)

  2. This search will take me to the details page of a specific bibliographic record where I can click on “Add to” and select “List” from the dropdown

  3. A pop-up window will appear – once you add a name and choose a category, click on “Save”

  4. The new list should now appear in the list of lists in the Lists module

Creating lists from the OPAC¶

Creating a list from “Your lists”¶

  1. Click on your account at nextkansas.org

  2. On the summary page, click on “Your lists”

  3. On the “Your lists” tab, click on “+ NEW LIST”

  4. Give the list a name; set the sort order; mark the list as “Private” or “Public;” and set the “Allow changes …” permissions and click on “Save”

  5. The catalog will take you to a “List created” message

Creating a list from a search¶

  1. Search for materials

  2. Check the checkboxes next to the titles you want to add to your list

  3. Click on the “With selected titles” dropdown and choose “New list”

  4. A pop-up window will appear – once you add a name and choose a category, click on “Save”

  5. Go to your account and click on “Your lists”

  6. The list should now appear in your list of lists
