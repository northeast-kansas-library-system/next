2021 Annual statistical data¶

2021 data¶

The 2021 data can be found on the Next Search Catalog home page on the statistics tab.

This spreadsheet answers the following questions for the annual state statistical survey:

  1. Question 2.2a - Number of users
  • Data comes from the “Monthly statistics” spreadsheet “YTD totals tab” - column U.
  • Patrons whose accounts had been expired for more than 3 years and had no issues that would have prevented their accounts from being automatically deleted, were automatically deleted several times in 2021. This fulfills the “Inactive patron accounts should have been purged within the past three (3) years” note added to that question.
  • The report that gathers this data counts the number of patrons in the borrowers table in the system on January 1, 2022.
  • This year, the report was generated automatically between midnight and 2:00 a.m. on January 1, 2022.
  1. Question 9.1a - books owned at start of 2021
  • Data gathered from Report 2867.
  • This report counts items in the items table based on item type and collection code that were in the database on January 1, 2022.
  • This report has changed dramatically since last year due to changes in item types and in collection codes. In the past many items were counted twice due to problems with the report.
  • This year, the report was generated automatically between midnight and 2:00 a.m. on January 1, 2022.
  1. Question 9.1b - books added during 2021
  • Data gathered from Report 2868.
  • The “BOOKS_A” column of this report counts items in the items and deleted items tables based on item type and collection code where the item’s dateaccessioned field shows 2021 as the year. By counting items in the deleted items table, this report takes into account items that may have been added and deleted during 2021 (i.e. if a book was added in January, was then checked out and kept long overdue, was later marked lost, and was eventually deleted in June, that book will still be counted as being added in 2021).
  • This report has changed dramatically since last year due to changes in item types and in collection codes. In the past many items were counted twice due to problems with the report.
  • This year, the report was generated automatically between midnight and 2:00 a.m. on January 1, 2022.
  1. Question 9.1c - books deleted during 2021
  • Data gathered from Report 2868.
  • The “BOOKS_D” column of this report counts items that were added to the deleted item table based on item type and collection code where the deleted item’s timestamp shows 2021 as the year.
  • This report has changed dramatically since last year due to changes in item types and in collection codes. In the past many items were counted twice due to problems with the report.
  • This year, the report was generated automatically between midnight and 2:00 a.m. on January 1, 2022.
  1. Question 9.2 - total audio at end of 2021
  • Data gathered from Report 2868.
  • The “AUDIO_T” column of this report counts items in the items table by item type (if the item is one of the audiobook item types) or by collection code (if the item type is not an audiobook item type but the collection code is one of the audiobook collection codes).
  • This report has changed dramatically since last year due to changes in item types and in collection codes. In the past many items were counted twice due to problems with the report.
  • This year, the report was generated automatically between midnight and 2:00 a.m. on January 1, 2022.
  1. Question 9.3 - total video at end of 2021
  • Data gathered from Report 2868.
  • The “VIDEO_T” column of this report counts items in the items table by item type (if the item is one of the video item types) or by collection code (if the item type is not a video item type but the collection code collection code is one of the video item types).
  • This report has changed dramatically since last year due to changes in item types and in collection codes. In the past many items were counted twice due to problems with the report.
  • This year, the report was generated automatically between midnight and 2:00 a.m. on January 1, 2022.
  1. Question 9.4 - total of all other materials at end of 2021
  • Data gathered from Report 2868.
  • This data is generated by taking the total of all items and subtracting the results of questions 9.1b, 9.1c, 9.2, and 9.3a as determined by the other queries in this report.
  • This report has changed dramatically since last year due to changes in item types and in collection codes. In the past many items were counted twice due to problems with the report.
  • This year, the report was generated automatically between midnight and 2:00 a.m. on January 1, 2022.
  1. Question 11.1 - circulation of adult materials
  • Data comes from the “Monthly statistics” spreadsheet “YTD totals tab” - column P
  • Data to build this answer was calculated monthly in the “2021 calendar year statistics” spreadsheet to make sure the data was as accurate as possible.
  1. Question 11.2 - circulation of non-adult materials
  • Data comes from the “Monthly statistics” spreadsheet “YTD totals tab” - column Q
  • Data to build this answer was calculated monthly in the “2021 calendar year statistics” spreadsheet to make sure the data was as accurate as possible.
  1. Question 11.4 - circulation of “Other physical items”
  • New question for 2021. Since this question is new, there is no data from 2021 to answer the question accurately.
  • The goal here appears to be to separate content from what we’ve been doing in the past (i.e. “Print materials” vs. “Non-print materials” vs. “Electronic material”) by adding a new category for physical materials that could best be described as “Realia.” In other words, Cake pans; tools; bicycles; WiFi hotspots; etc. – physical materials where the item is the content rather than a book or a DVD where the item contains the content.
  • Steps will be taken to answer this question for 2022
  • See `https://github.com/IMLS/public-libraries-survey/blob/master/FY%202021%20PLS%20Data%20Elements.pdf`_ for more information
  1. Question 11.10 - NExpress ILLs borrowerd from other NExpress libraries
  • Data comes from the “Monthly statistics” spreadsheet “YTD totals tab” - column Y
  • Data to build this answer was calculated monthly in the “2021 calendar year statistics” spreadsheet to make sure the data was as accurate as possible.
  1. Question 11.14 - NExpress ILLs loaned to other NExpress libraries
  • Data comes from the “Monthly statistics” spreadsheet “YTD totals tab” - column Z
  • Data to build this answer was calculated monthly in the “2021 calendar year statistics” spreadsheet to make sure the data was as accurate as possible.
